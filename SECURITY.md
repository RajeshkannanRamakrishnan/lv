## Reporting a Vulnerability

We take the security of our software seriously. If you believe you have found a security vulnerability in `lv`, please report it to us as described below.

**Please do not report security vulnerabilities through public GitHub issues.**

### How to Report

Please send an email to the project maintainers. If you are comfortable doing so, please use PGP to encrypt your email.

You should receive a response within 48 hours. If for some reason you do not, please follow up via email to ensure we received your original message.

### What to Include

*   Detailed steps to reproduce the vulnerability (POC scripts, screenshots, etc.).
*   The version of `lv` you were running.
*   The operating system and architecture.

### Policy

*   We will acknowledge your report within 48 hours.
*   We will provide an estimated timeframe for addressing the vulnerability.
*   We will notify you when the vulnerability is fixed.
*   We will credit you for the discovery (if you wish) in our release notes.

Thank you for helping keep `lv` safe!
